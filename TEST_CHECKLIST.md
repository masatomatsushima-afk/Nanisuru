# Nanisuru リリース前テストチェックリスト

手動テスト用チェックリストです。リリース前または大きな機能追加後に、実機またはシミュレータで確認してください。

## 前提

- [ ] `.env` に API キーと Supabase 設定がある（`.env` は git に含めない）
- [ ] Supabase SQL が必要な順序で実行済み（`.env.example` 参照）
- [ ] `npx expo start --clear` でアプリを起動できる

---

## 1. 認証

- [ ] 未ログインで保存済みプラン詳細を開く → ログインプロンプトが表示される
- [ ] 未ログインで通知画面を開く → ログインプロンプトが表示される
- [ ] 未ログインでプランコピー編集を開く → ログインプロンプトが表示される
- [ ] Google / Apple ログインが成功する
- [ ] ログイン後、保存済みプラン・通知にアクセスできる
- [ ] ログアウト後、プライベート画面が保護される

---

## 2. プラン作成・保存

- [ ] ホームでプランを生成できる（ローディング表示あり）
- [ ] OpenAI 未設定時に分かりやすいエラーが出る
- [ ] 生成したプランを保存できる
- [ ] 保存済みタブにプランが表示される
- [ ] 保存済みプランが0件のとき空状態が表示される
- [ ] 保存済みプラン詳細を開ける

---

## 3. 公開プラン

- [ ] 保存済みプランから「このプランを公開する」が使える
- [ ] 公開中ローディング（「公開中...」）が表示される
- [ ] 画像を追加して公開できる
- [ ] 動画リンクを追加して公開できる
- [ ] 発見タブに公開プランが表示される
- [ ] 公開プラン詳細を開ける
- [ ] クリエイターが「公開を停止する」できる
- [ ] クリエイターが「公開プランを削除」できる（確認ダイアログあり）

---

## 4. ソーシャル操作

- [ ] 公開プランにいいねできる（「いいね中...」表示）
- [ ] 公開プランを保存できる（「保存中...」表示）
- [ ] ユーザーをフォローできる（「処理中...」表示）
- [ ] コメントを投稿できる
- [ ] コメントが0件のとき空状態が表示される
- [ ] 未ログイン時、上記操作でログイン誘導が出る

---

## 5. コピー・AIバージョン

- [ ] 「このプランをコピーして編集」が動作する（「コピー中...」）
- [ ] コピー後の編集画面で保存できる
- [ ] AI調整が動作する
- [ ] 改善リクエストから AI バージョンを作成できる（「生成中...」）
- [ ] 下書き保存・公開ができる

---

## 6. 通知

- [ ] マイページから通知画面を開ける
- [ ] いいね・保存・コメント・フォローで通知が届く
- [ ] 通知タップでプランまたはプロフィールに遷移する
- [ ] 「すべて既読にする」が動作する
- [ ] 通知が0件のとき「まだ通知はありません」が表示される

---

## 7. 安全・モデレーション

- [ ] プランを通報できる
- [ ] コメントを通報できる
- [ ] ユーザーをブロックできる
- [ ] ブロック後、そのユーザーのプラン・コメントが非表示になる
- [ ] ブロックユーザーの通知が表示されない

---

## 8. 発見・検索

- [ ] おすすめセクションが表示される
- [ ] 「AIに探してもらう」が動作する
- [ ] フィルター・検索で結果が絞り込める
- [ ] 条件に合うプランがないとき空状態が表示される
- [ ] 公開プランが0件のとき空状態が表示される

---

## 9. エラー・オフライン

- [ ] 機内モードで Supabase 操作 → エラーメッセージと再試行が表示される
- [ ] 位置情報拒否時に適切なメッセージが出る
- [ ] Places 取得失敗時にプラン生成がフォールバックまたはエラー表示される

---

## 10. ナビゲーション

- [ ] 行程から Google Maps / 道順が開ける
- [ ] 共有リンクがコピーまたは共有できる
- [ ] 公開プラン → クリエイタープロフィール → 戻る が正しく動く
- [ ] タブ間の遷移が崩れない

---

## 11. セキュリティ確認

- [ ] 公開プロフィールにメールアドレスが表示されない（表示名のみ）
- [ ] 他人の非公開プランにアクセスできない
- [ ] `.env` が git にコミットされていない
